### 2. Chức năng

* Tạo đơn hàng
* Cập nhật trạng thái đơn
* Xem đơn hàng

### 3. Actor

* User, Admin

1. User chọn món
2. Thêm vào giỏ

4. Hệ thống tạo đơn

### 5. Luồng phụ

* Giỏ hàng trống → không cho đặt

### 6. Trạng thái đơn
