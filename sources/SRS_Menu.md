## SRS – Module Menu (Món ăn)

### 1. Mô tả

Quản lý danh sách món ăn.

### 2. Chức năng

* Hiển thị danh sách món
* Xem chi tiết món
* Tìm kiếm món
* Thêm / sửa / xóa món (Admin)

### 3. Actor

* User, Admin

### 4. Luồng chính

1. User truy cập menu
2. Hệ thống hiển thị món
3. User chọn món

### 5. Ràng buộc

* Món phải thuộc danh mục
* Có giá và hình ảnh
