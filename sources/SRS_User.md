## SRS – Module User

### 1. Mô tả

Quản lý thông tin người dùng và các thao tác liên quan đến tài khoản.

### 2. Chức năng

* Đăng ký tài khoản
* Đăng nhập / đăng xuất
* Cập nhật thông tin cá nhân
* Đổi mật khẩu
* Xem lịch sử đơn hàng

### 3. Actor

* User

### 4. Luồng chính

1. User truy cập hệ thống
2. Thực hiện đăng ký hoặc đăng nhập
3. Quản lý thông tin cá nhân
4. Theo dõi đơn hàng

### 5. Ràng buộc

* Email không được trùng
* Mật khẩu phải được mã hóa
